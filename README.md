# TMSTH Prem-Document

Repository นี้ใช้สำหรับจัดเก็บเอกสาร Markdown ของระบบงานต่าง ๆ และแสดงผลผ่าน GitHub Pages

## Structure

- `index.html` หน้าเว็บหลัก
- `style.css` ไฟล์ตกแต่งหน้าจอ
- `app.js` ไฟล์โหลด Markdown
- `docs/` โฟลเดอร์เก็บเอกสารทั้งหมด

## Update Document

กรณีแก้เอกสารเดิม:
1. แก้ไฟล์ `.md`
2. Commit
3. Push

กรณีเพิ่มเอกสารใหม่:
1. เพิ่มไฟล์ `.md` ใน `docs/`
2. เพิ่มเมนูใน `index.html`
3. Commit
4. Push
